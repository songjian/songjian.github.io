## 安装

最新稳定版

[https://github.com/neovim/neovim/releases/tag/stable](https://github.com/neovim/neovim/releases/tag/stable)

安装

```sh
sudo apt install ./nvim-linux64.deb
```

## 配置

创建配置文件目录

```sh
mkdir -p ~/.config/nvim
```

创建配置文件

```sh
touch ~/.config/nvim/init.lua
```

:so 重新载入配置
