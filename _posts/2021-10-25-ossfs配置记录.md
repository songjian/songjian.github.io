## 步骤

### 1. 从github获取代码编译

```sh
git pull git@github.com:aliyun/ossfs.git
cd ossfs
./autogen.sh
./configure
make
sudo make install
```

### 2. 配置Bucket

```sh
echo my-bucket:my-access-key-id:my-access-key-secret > /etc/passwd-ossfs
chmod 640 /etc/passwd-ossfs
```

### 3. ossfs绑定目录

```sh
ossfs my-bucket my-mount-point -ourl=my-oss-endpoint
```
