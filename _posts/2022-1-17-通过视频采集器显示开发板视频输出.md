### 软件

#### 安装OBS Studio

[OBS Studio](https://obsproject.com/)

```sh
sudo add-apt-repository ppa:obsproject/obs-studio
sudo apt update
sudo apt install ffmpeg obs-studio
```

#### 安装视频采集卡驱动

```sh
sudo apt install v4l2loopback-dkms
```
