[gettext](https://www.gnu.org/software/gettext/)是自由软件国际化的事实标准。

### 文件说明

* `pot` 是模板文件
* `po` 给翻译人员用的语言文件
* `mo` 二级制语言文件，给程序读取用

### 安装

```sh
# Ubuntu
apt install gettext
```

### 使用

#### 从 pot 制作 po

```sh
msginit -i file.pot
```

### 编程

```sh
printf(gettext("My name is %s.\n"), my_name);
printf(_("My name is %s.\n"), my_name); // same, but shorter
```
