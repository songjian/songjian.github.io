> https://github.com/Genymobile/scrcpy

[Scrcpy](https://github.com/Genymobile/scrcpy)是一个开源的Android屏幕复制软件。

### 安装

```sh
sudo apt install scrcpy
```

### adb查看设备

```sh
adb devices
```

### 运行

```sh
scrcpy
```
